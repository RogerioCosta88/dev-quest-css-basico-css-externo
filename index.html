CSS Externo

O CSS Externo é a melhor forma de estilizar um site, porque podemos aplicar o mesmo estilo em várias páginas diferentes ao mesmo tempo e nas outras formas de estilização CSS Interno e Inline não é possível fazer isso.

No CSS Externo utilizamos um documento no formato .css e lincamos com o HTML para aplicar os estilos.

Todas as páginas HTML são lincadas com esse documento de CSS, fazendo com que o estilo criado para os paragráfos sejam feitos uma única vez.
